диалог для установки процентов скидки для групп скидок на папку или товар

Процедура поиска цены для товара
I. Цена формируется на основании прайс листа (по умолчанию)
	1. Вид цены из прайс листа. Это основа для формирования цены. Выбирается из свойств получателя/отправителя/моя фирма
	2. Дальше работаем либо по группе скидок, либо по скидке на сумму, в зависимости от того, что установлено у получателя
		2.1. Акционный товар. Цена для акционного товара выставляется в отдельной колонке прайс-листа, не зависит от всех остальных вариантов скидки и не участвует в расчете скидки на документ
		2.2. Группы скидок. Если для получателя установлена группа скидок, то цены из п.1 корректируются в соответствии с уровнем скидки
		2.3. Накопительная скидка (дисконт). Считается от суммы покупок клиента за определенный период
		2.4. Скидка на сумму документа. Если для получателя предусмотрена скидка на сумму, то считаем ее. Для этого нужно еще одну проводку добавить
II. Цена формируется из свойств партии
III. Цена формируется как процент наценки на среднюю цену себестоимости. Процент наценки берется из свойств товара (родителя)


Решение
1. Для акционной скидки создаем вид цены и запоминаем его в параметрах БД
2. для управления значениями скидок для группы добавляем диалог для товара (папки/группы). Значение скидок храним в нормах.
