Версия 007 (24 ноября 2015)

Функциональность
  • Добавлена компонента Кассовый регистратор  для печати фискальных чеков на кассовых регистраторах.
  • Магазин. Добавлен функционал для поддержки кассового регистратора: формы первичных документов для вноса и вывода денежных средств в кассу.
  • Магазин. Реализация товара. Добавлена кнопка Чек продажи для печати чека продажи.
  • Магазин. Возврат от покупателя. Добавлена кнопка Чек возврата для печати чека возврата.
  • Услуги. Оказанные. Добавлена кнопка для печати чека продажи и чека возврата.
  • В документы по группы Товар и Материалы добавлена кнопка Подобрать для отображения диалога с остатками товара и возможностью его и выбора.
  • Добавилась возможность устанавливать процент наценки на родительской папки объекта учета. Он используется при формировании цены на поступивший товар. Для установки процента наценки требуется указать значение одноименного параметра папки.
  • В разделе Разное добавлена ветка Торговое оборудование для регистрации и управления торговым оборудованием, подключаемым к программе.
