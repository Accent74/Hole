Версия 004 (3 сентября 2015)

Содержание <<Предыдущая Следующая>>  
Исправленные ошибки
  • Формы первичных документов. Исправлена ошибка, которая возникала при попытке создать новый документ на основании текущего.
  • Модуль списания. Исправлена ошибка, из-за которой ошибочно выдавалась сообщение об отсутствии партии товара.
  • Модуль работы с XML файлами. Исправлена грамматическая ошибка в тексте сообщения.
  • Авансовый отчет. Исправлена страница для печати документа.
  • Бухгалтерская справка. Исправлена ошибка, из-за которой вместо счета по Дт проводки отображалось разное.
  • Счет исходящий. Исправлена ошибка, из-за которой не проставлялась цена из прайс-листа.
  • Накладная на поступление материалов. Исправлена ошибка, из-за которой в многострочном документе не наследовалась валюта.
  • Расходная накладная (товар). Исправлена ошибка на странице печати документа формата А4.
  • Ведомость обязательных удержаних из заработной платы. Исправлена ошибка, из-за которой она не заполнялась данными.
  • Шаблоны. Исправлен шаблон для начисления заработной платы. В нем по Дт установлен счет 91. Если требуется отражать затраты по подразделениям на отдельных счетах, можно использовать одноименный параметр подразделения.
  • Шаблоны. Исправлен шаблон для обязательных начислений на заработную плату. В формуле для расчета суммы по операции ошибочно стояла ссылка на вторую проводку, которой нет.
Функциональность
  • Товарный отчет. Добавлен товарный отчет, который позволяет оценить движение товара, денег и рассчитать прибыль.
  • Ведомость начисления заработной платы. Добавились ограничения на выбор подразделения (только в пределах Моя компания) и выборе сотрудника (только в пределах выбранного подразделения).
  • Ведомость начисления заработной платы. В процессе выбор сотрудников проводится проверка на дубликаты.
  • Добавлена форма для выплаты заработной платы
База даных
  • Шаблоны. Добавлены шаблоны для выплаты заработной платы.
  • Шаблоны. Добавлены шаблоны для ввода вступительного баланса.
  • Папки документов. Добавлены папки документов для выплаты заработной платы.
  • Папки первичных документов. Добавлены первичных документов для ввода вступительного баланса.
